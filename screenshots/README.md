## How to take screenshots

1. Run the frontend: `cd frontend && npm run dev`
2. Open the app in your browser
3. Press `F11` for fullscreen mode
4. Use `Ctrl+Shift+S` (Edge) or your preferred screenshot tool
5. Save images with the names above
